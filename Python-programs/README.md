# Python Programs

This folder contains basic Python programs created for practice while learning Python fundamentals.

## Programs Included

- **Greater.py**  
  A simple program to compare values and find the greater number.

- **fruits.py**  
  A basic program related to fruits (list / input / output practice).

- **spc_arena.py**  
  A practice program created for SPC Arena tasks and logic building.

## Purpose

- Practice Python basics
- Improve logic building
- Learn file structure and GitHub workflow

## How to Run

Use Python to run any file:
python filename.py
